Gameplay Tutorial

The aim of the game is to catch the required Pokemon before running out of Pokeballs.

If you have caught the required Pokemon for a level, you may proceed to the next level.

If you have not caught the required Pokemon, you must restart the level.

Controls

Use the "WASD" keys to move

Use "Space-bar" to jump

Hold "Space-bar" to double jump

Use "Left Shift" to run

Click on the Pokemon to throw a Pokeball and catch them

Use "M" for the Minimap

Use "P" for the Pause Menu

Click on side menu to see the Tutorial
